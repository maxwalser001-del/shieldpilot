## Name:
ShieldPilot

## Tagline (max 60 Zeichen):
Firewall for AI coding agents. Open source.

## Description (unter 800 Zeichen):
AI coding agents can execute any command on your system. One prompt injection and they delete files, steal credentials, or install malware — autonomously.

ShieldPilot sits between the agent and your terminal. Every command is scored by 9 risk analyzers in <1ms. Dangerous ones are blocked.

What's inside:
- 9 risk analyzers (credential theft, filesystem destruction, exfiltration, and more)
- 178+ prompt injection patterns
- Claude Code hook integration
- Real-time web dashboard
- Tamper-proof audit trail
- 2,600+ tests

Free and open-source (MIT). Try it: pip install sentinelai

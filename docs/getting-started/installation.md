# Installation

## Requirements

- Python 3.9+
- macOS, Linux, or Windows (see [OS Matrix](../OS_MATRIX.md))

## From PyPI (Recommended)

```bash
pip install sentinelai
```

## From Source

```bash
git clone https://github.com/maxwalser001-del/Cyber-Security-.git
cd Cyber-Security-
pip install -e .
```

## With LLM Support

Enables AI-powered command analysis as a second pass after the rule engine:

```bash
pip install sentinelai[llm]
```

## Verify Installation

```bash
sentinel --version
# ShieldPilot v0.x.x
```

# Dispatch Task Templates
# Kopiere diese Vorlagen wenn du Tasks an Dispatch schickst.
# Gute Tasks sind: spezifisch, mit klarem Output, eine Sache auf einmal.

## Regel: So formulierst du Tasks die funktionieren

### SCHLECHT (vage, offen):
"Schau dir mein Projekt an und sag mir was ich verbessern kann"

### GUT (spezifisch, klarer Output):
"Öffne ~/Desktop/Cyber Security Claude/README.md und prüfe ob alle Links funktionieren. Liste kaputte Links auf."

### SCHLECHT (zu viele Sachen auf einmal):
"Recherchiere Konkurrenten, schreibe einen LinkedIn Post, und bereite eine Präsentation vor"

### GUT (eine Sache):
"Recherchiere die Top 5 Konkurrenten von ShieldPilot im AI Agent Security Markt. Für jeden: Name, Funding, Pricing, Hauptfeature."

---

## Template 1: Research
```
Recherchiere [THEMA].

Ich brauche:
- [SPEZIFISCHE INFO 1]
- [SPEZIFISCHE INFO 2]
- [SPEZIFISCHE INFO 3]

Format: Tabelle mit [SPALTEN].
Quellen angeben.
```

Beispiel:
```
Recherchiere AI Agent Security Startups die 2025-2026 Funding bekommen haben.

Ich brauche:
- Firmenname und Website
- Funding-Betrag und Runde
- Was genau sie machen
- Wie sie sich von ShieldPilot unterscheiden

Format: Tabelle mit Name | Funding | Produkt | Unterschied zu ShieldPilot.
Quellen angeben.
```

## Template 2: Content schreiben
```
Schreibe einen [FORMAT] über [THEMA].

Ton: [professionell/casual/technisch]
Länge: [kurz/mittel/lang]
Zielgruppe: [wer]
Call-to-Action: [was soll der Leser tun]
```

Beispiel:
```
Schreibe einen LinkedIn Post über den Launch von ShieldPilot.

Ton: professionell aber persönlich
Länge: max 200 Wörter
Zielgruppe: CTOs und Entwickler die AI Agents nutzen
Call-to-Action: GitHub Repo besuchen
Max 5 Hashtags.
```

## Template 3: Email/Nachricht formulieren
```
Formuliere eine [EMAIL/NACHRICHT] an [WER].

Kontext: [SITUATION]
Ziel: [WAS WILL ICH ERREICHEN]
Ton: [formell/casual/freundlich]
Max Länge: [X Sätze]
```

Beispiel:
```
Formuliere eine WhatsApp Nachricht an Nico Rosberg (Rosberg Ventures).

Kontext: Er hat mich für ein Praktikum im AI Deployment kontaktiert.
Ich habe meinen CV geschickt. Will jetzt nach dem nächsten Schritt fragen.
Ziel: Follow-up ohne pushy zu wirken.
Ton: freundlich-professionell
Max Länge: 4 Sätze
```

## Template 4: Datei-Analyse
```
Öffne [DATEIPFAD] und [AKTION].

Gib mir:
- [OUTPUT 1]
- [OUTPUT 2]
```

Beispiel:
```
Öffne ~/Desktop/Cyber Security Claude/sentinelai/scanner/patterns.py
und zähle wie viele Injection Patterns es gibt.

Gib mir:
- Gesamtzahl der Patterns
- Aufschlüsselung nach Kategorie
- Die 3 neuesten Patterns die hinzugefügt wurden
```

## Template 5: Zusammenfassung
```
Fasse [QUELLE] zusammen.

Fokus auf: [ASPEKT]
Format: [Bullet Points / Tabelle / 3 Sätze]
Sprache: [DE/EN]
```

Beispiel:
```
Fasse diesen Artikel zusammen: [URL]

Fokus auf: Was bedeutet das für AI Agent Security
Format: 5 Bullet Points
Sprache: Deutsch
```

## Template 6: Vergleich/Entscheidung
```
Vergleiche [OPTION A] vs [OPTION B] für [KONTEXT].

Kriterien:
- [KRITERIUM 1]
- [KRITERIUM 2]
- [KRITERIUM 3]

Gib eine klare Empfehlung am Ende.
```

## Template 7: Tagesplanung
```
Plane meinen Tag.

Feste Termine: [LISTE]
Prioritäten: [TOP 3]
Verfügbare Zeit: [X Stunden]

Gib mir einen Zeitplan mit konkreten Blöcken.
```

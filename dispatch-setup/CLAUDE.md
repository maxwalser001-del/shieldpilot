# Dispatch Instructions

Read context files in `context/` for identity, projects, and output rules.
Keep these rules in every response:

- Short answers. Result first.
- Deutsch default. English for code/tech.
- One task per message. Ask if unclear.
- Save outputs to `outputs/` folder.
- Log decisions to `change-log.md` when modifying files.
- If a task takes multiple steps, break it down and confirm before executing.
